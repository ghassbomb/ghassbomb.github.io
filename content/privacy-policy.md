## Cookies

This website uses cookies. By using this website and agreeing to this policy, you consent to **ghassbomb.github.io**’s use of cookies in accordance with the terms of this policy. Cookies are information packets sent by web servers to web browsers, and stored by web browsers. The information is then sent back to the server each time the browser requests a page from the server. This enables a web server to identify and track web browsers.

In order to comply with your preferences, **ghassbomb.github.io** may store a small number tiny functional cookies so that you’re not  asked to make this choice again, to be able to save your settings and to warn you of an updated changelog. Please note that these cookie are only used to guarantee a user-friendly visit and a proper website performance.
